# Environment

macOS Sierra 10.12.1

# Installation

```
git clone git@github.com:kimochg/dotfiles.git ~/dotfiles && cd ~/dotfiles/setup && chmod a+x setup.sh && ./setup.sh
```

# Setup

1. open `tmux`, use `CTRL + A, I` install plugins
2. open `macvim`, use `:PluginInstall`
